# Lanzou-Worker-Parser
Inspired by [hanximeng](https://github.com/hanximeng/LanzouAPI/commits?author=hanximeng)'s PHP [LanzouAPI](https://github.com/hanximeng/LanzouAPI/tree/master)
## 用法
直接在Cloudflare Workers部署workers.js即可，完全免费
## 功能
1. 支持检测文件是否被取消
2. 支持带密码的文件分享链接但不支持分享的文件夹
3. 支持生成直链或直接下载
4. 增加ios应用在线安装
5. 解析最终直链
6. 自动识别旧版链接替换为新版并解析
## 使用方法
url:蓝奏云外链链接
type:是否直接下载 值：down
pwd:外链密码
## 调用示例
我们不提供示例demo，请自行部署后使用。
https://lanzou.api.demo/?url=https://download.lanzout.com/ithiaaka6qnc&pwd=1234&down=true
